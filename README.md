# Vertex AI-Integrated ML Pipeline with Apache Beam

## Overview
Developed a predictive analytics pipeline by combining Apache Beam and Vertex AI, reducing model retraining time by 50%. Deployed workflows for real-time inference using BigQuery.

## Key Features
- Scalable pipeline with Apache Beam for batch and streaming data.
- Integrated Vertex AI for model training and deployment.
- Real-time inference with BigQuery.

## Tools & Technologies
- Apache Beam
- Vertex AI
- BigQuery

## Results
- Reduced model retraining time by 50%.
- Achieved a 20% improvement in prediction accuracy.
